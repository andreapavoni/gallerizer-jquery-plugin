jQuery Gallerizer v0.2 demo

Step 1

Include required js files. 

        <script src="jquery-1.3.2.min.js" type="text/javascript"></script>

        <script src="jquery.gallerizer-0.2.js" type="text/javascript"></script>

        <!-- ... other head tags ... -->

Step 2

Create an array of image filenames, then start gallerize on some img element in the page selected by class or id.

      var $photos = new Array('image_one.png','photo_2.png','etc_img.png');

      $(document).ready(function() {
        $('#demo1').gallerize($photos);
      });

Step 3

Add id or class you used to start gallerize() to img tag.

      <img src="image_one.png" id="demo1" alt="demo1" />
